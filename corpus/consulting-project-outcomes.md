Client-facing delivery from Alex's Deloitte Digital consulting years — three public-sector and care-sector Salesforce engagements, each with a different delivery shape.

**NSW Police Force — public assembly form digitisation (Dec 2022 – Apr 2023).** Main developer digitising the Notice to Hold a Public Assembly — replacing paper forms with an LWC digital form embedded in the force's existing Salesforce Experience Cloud community portal. Built the LWC form body integrating existing components (Google reCAPTCHA, address lookup) and connecting to Salesforce custom objects. Deployed 180+ components to production via Change Set deployment, managed git feature branches and pull requests throughout, then ran smoke testing, post-deployment validation, and on-site/remote demo workshops with NSW Police Force stakeholders.

**WA Health — vaccine management system (Jan 2022 – Aug 2022).** Joined the CI/CD stream to design and build user provisioning automation, reducing manual effort from 23 minutes to 3 minutes per user account. Delivered Apex Email Service handlers with test classes, Permission Set Groups, custom objects and metadata, and 7 Flows covering distinctive system features — managing feature branches, pull requests, merge conflict resolution, and deployment packages across the project lifecycle.

**ACH Group — aged-care Salesforce implementation (Jun 2021 – Nov 2021).** Full-lifecycle involvement from mobilisation and scope definition through go-live support. Facilitated client workshops, prepared business process maps, and translated requirements into user stories and acceptance criteria with BAs and stakeholders. Migrated data from the legacy system via Salesforce Data Loader — cleaning, transformation, import execution, and migration reporting — and performed system testing across all Salesforce-related user stories covering technical implementation, third-party app integration, and data migration.
